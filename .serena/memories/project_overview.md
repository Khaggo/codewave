AUTOCARE is a Windows-based monorepo/workspace centered on backend source-of-truth architecture docs plus a TypeScript/NestJS backend implementation and a lightweight frontend contract workspace. The backend is split into `main-service` and `ecommerce-service` NestJS apps under `backend/apps`. Canonical business/architecture truth lives under `docs/architecture`, with non-canonical implementation tasks under `docs/architecture/tasks` and frontend/backend handoff packs under `docs/contracts`. The `frontend` folder is not yet a full Next.js application; it currently holds slice-based typed API contracts in `frontend/src/lib/api/generated` and mock fixtures in `frontend/src/mocks` for frontend-first or parallel slice work.
